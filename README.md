# jds0295's NVIM setup

## How to setup

- Install Packer [Github](https://github.com/wbthomason/packer.nvim) by running: `git clone --depth 1 https://github.com/wbthomason/packer.nvim\
 ~/.local/share/nvim/site/pack/packer/start/packer.nvim`
- once installed in nvim run `:PackerSync` to install all plugins or run `$ nvim --headless -c 'autocmd User PackerComplete quitall' -c 'PackerSync'`
